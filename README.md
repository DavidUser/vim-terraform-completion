# Vim Terraform Completion

**A very early prototype**

## Installation

**Require `+ruby` or `+ruby/dyn` for vim**

### NeoBundle
`NeoBundle 'juliosueiras/vim-terraform-completion'`

### Vim-Plug
`Plug 'juliosueiras/vim-terraform-completion'`
